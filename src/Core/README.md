# Defra.Lis.Core

A core library containing common functionality shared across Defra .NET projects.

## Contents

- **`Constants`** — shared constant values (e.g. proxy environment variable names).
- **`Exceptions`** — common exception types:
  - `BusinessRuleException` — thrown when a business rule is violated.
  - `ConflictException` — thrown when a conflict occurs.
  - `ProxyException` — thrown when a proxy error occurs.

## Installation

```sh
dotnet add package Defra.Lis.Core
```
